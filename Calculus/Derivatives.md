* Standard Derivatives
  * $(x^r)' = rx^{r-1}$
  * $(e^x)' = e^x$
  * $(ln\ x)' = \frac{1}{x}$
  * $(\sin(x))' = \cos(x)$
  * $(\cos(x))' = -\sin(x)$
* Computation Rules
  * $(cf(x))'=cf'(x)\hspace{2cm}\text{for any real value } c$
  * $(f(x)+g(x))'=f'(x)+g'(x)$
  * $(f(x)g(x))'=f'(x)g(x)+f(x)g'(x)$
  * $\left(\frac{f(x)}{g(x)}\right)' = \frac{f'(x)g(x)-f(x)g'(x)}{(g(x))^2}$
  * $f(g(x))' = f'(g(x))g'(x)$
